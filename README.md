# BumbleSMM

The World's #1 SMM Panel for Viral Growth.

## Features

- **Facebook First**: Prioritized for Facebook services.
- **Real-time Notifications**: Sound and toast notifications for admins and users.
- **Admin Dashboard**: Comprehensive service and user management.
- **User Dashboard**: Easy order placement and fund management.
- **Modern UI**: Built with React, Tailwind CSS, and Shadcn UI.

## Technology Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase (Mock DB for Demo)

## Setup

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
